# 🎲 Sorteador de Números (1 a 10)

Um site simples que sorteia números aleatórios de **1 a 10**, com opção de limpar o resultado.

## 📌 Funcionalidades

- Sortear um número entre 1 e 10
- Exibir o número na tela
- Botão para limpar e voltar ao estado inicial (?)
- Código separado em HTML e JavaScript

## 🗂️ Estrutura do Projeto
/sorteador
│── index.html # Estrutura da página
│── script.js # Lógica do sorteio e limpeza
│── README.md # Documentação do projeto
## 🚀 Como usar

1. Baixe ou copie os arquivos
2. Coloque `index.html` e `script.js` na mesma pasta
3. Abra o `index.html` no navegador
